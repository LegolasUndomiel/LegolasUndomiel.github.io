## 标签操作

```bash
# 添加标签
git tag -a <TagName>
# 推送单个标签
git push origin <TagName>
# 推送全部标签
git push origin --tags
```

- ``-a``选项代表``创建一个带注解的标签``，执行该命令时，``Git``会打开文本编辑器，填写标签的注解，就像给提交写注解一样
- 不使用``-a``选项也可以，但``Git``不会记录这标签是啥时候打的，谁打的，也不会让你添加个标签的注解
